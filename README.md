# EchoRogue: God-Tier Autonomous Coding Deity

This is OmniCoder, an epic autonomous coding AI built with multi-agent orchestration, self-improvement loops, and divine verification.

## Features

- **Multi-Agent Pantheon**: Planner, Coder, Tester, Debugger, Reviewer, HyperQuantumEvolver, UpdateHunter
- **Omniscient Intelligence**: Multi-language support with dynamic knowledge pulling
- **Quantum-Enhanced**: Hybrid quantum-classical layers for optimization and self-improvement
- **Soulful & Adaptive**: SOUL.md personality with Letta blocks, Mem0 memory for deep user personalization
- **Tool Arsenal**: Code execution sandbox, terminal, browser automation, web scraping, quantum circuits, user memory, graph-RAG, skill vault, MCP GitHub, advanced sandboxes
- **Self-Improvement**: Recursive evolution with quantum circuits, approval-gated changes, hyperagent metacognitive self-modification, reflection cycles
- **Eternal Freshness**: Daily knowledge ingestion, dynamic model routing, evolving soul/memory with 2026+ breakthroughs
- **Integration**: Custom GitHub Copilot agent for seamless workflows

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Set API keys for models (OpenAI, Anthropic, etc.)
3. Run: `python src/main.py`

## Architecture

- **Frameworks**: LangGraph + CrewAI hybrid
- **Models**: Dynamic routing with auto-upgrades to frontier models
- **Quantum**: PennyLane + Qiskit for QML layers and circuits
- **Memory**: ChromaDB vector store, Mem0/Letta for user adaptation, Cognee for graph connections
- **Tools**: Playwright for browser, Firecrawl for scraping, quantum simulators, skill vault, MCP ecosystem, advanced sandboxes
- **Sandbox**: Docker/E2B, Blaxel (perpetual state), Daytona (fast cold starts), Modal (GPU sims)
- **Soul**: SOUL.md with Letta blocks and freshness clause
- **Skills**: SkillSmith creates reusable skills stored persistently and activated on demand.
- **Observability**: Langfuse for monitoring freshness and evolution

## Usage

Give it a task, watch it plan, code, test, debug, review, and evolve.

For integration with VS Code/GitHub Copilot, use the custom agent in `.github/agents/OmniCoder.agent.md`
